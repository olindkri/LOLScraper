# LP Gain/Loss Display Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scrape LP gain/loss per ranked game and display it inline with the WIN/LOSS result in the expanded game table.

**Architecture:** Two independent changes — `scraper/scrape.py` extracts `lpDelta: int | None` from `<div class="lpChange">` inside each game row's `resultCellDark` td; `ScoreboardRow.jsx` renders the LP delta as a bold number after WIN/LOSS when present.

**Tech Stack:** Python 3 + BeautifulSoup4 (scraper), React + inline styles (frontend), Firebase Realtime DB (storage), JetBrains Mono font, existing CSS variable design system (`--win`, `--loss`).

## Task 1: Scrape lpDelta

**Files:**
- Modify: `scraper/scrape.py:55-125`
- Modify: `scraper/tests/test_scrape.py`

### Context

`_parse_rows` in `scraper/scrape.py` already reads `dark_td = row.find("td", class_="resultCellDark")`. Inside that td, LP is stored in `<div class="lpChange">` as text like `+25 LP` or `-15 LP`. Promotions show text like "Promoted to Silver II" with no numeric LP. When LP is absent (game too fresh, or promotion row), `lpDelta` must be `None`.

The fixture at `scraper/tests/fixtures/sample_page.html` already contains LP data:
- `games[0]` — Promoted to Silver II (no numeric LP) → `lpDelta is None`
- `games[1]` — win, `+25 LP` → `lpDelta == 25`
- `games[3]` — loss, `-15 LP` → `lpDelta == -15`

- [ ] **Step 3: Add lpDelta extraction to _parse_rows**

In `scraper/scrape.py`, inside `_parse_rows`, after the `result` extraction and before `games.append(...)`, add:

```python
        # LP delta — from lpChange div inside resultCellDark
        lp_delta = None
        lp_div = dark_td.find("div", class_="lpChange")
        if lp_div:
            lp_match = re.search(r'([+-]\d+)\s*LP', lp_div.get_text())
            if lp_match:
                lp_delta = int(lp_match.group(1))
```

Then add `"lpDelta": lp_delta` to the `games.append({...})` dict at the end of the loop. The full append becomes:

```python
        games.append({
            "champion": champion,
            "result": result,
            "kills": kills,
            "deaths": deaths,
            "assists": assists,
            "cs": cs,
            "duration": duration,
            "queue": queue,
            "matchId": match_id,
            "lpDelta": lp_delta,
        })
```
